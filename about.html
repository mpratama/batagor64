Fungsi App

Pada dasarnya app ini mempunyai 2 fungsi yaitu:

  1. Encode : Konversi file digital apapun kedalam bentuk teks menggunakan algoritma Base64.
  2. Decode : Konversi teks berformat Base64 ke bentuk filenya yang semula (gambar, dokumen, audio, video, dll).

Latar Belakang

Aplikasi ini ditulis berdasarkan pengalaman pembuat melaksanakan tugas negara di wilayah DTPK dimana pembangunan sarana komunikasi telekomunikasi dan transportasi belum mendapat porsi yang sama seperti di kota. Tiadanya jaringan internet menghambat komunikasi 2 arah dan berakibat menghambat kinerja. Adanya App ini bisa sedikit membantu menyelesaikan permasalahan tersebut. App ini bisa digunakan minimal jika masih ada layanan sms di wilayah DTPK. Dengan cara pengirim file mengkonversi (encoding) file digital (foto, file word, file excel, pdf, dll) kedalam bentuk Base64 teks lalu dikirim melalui sms. Penerima sms lalu mengkonversi kembali (decoding) Base64 teks ke bentuk file semula.

Pros & Cons

Pros:

  • Mengirim dan menerima file digital melalui jaringan sms.
  • Telah diuji (bukan di IPB & ITB) mengirim bermacam tipe file (gambar: jpeg, png, gif; dokumen: word, excel, pdf; program: java program. Secara teori dapat mengirim format file apapun.

Cons:

  • Mengirim teks lewat sms memiliki keterbatasan jumlah karakter. Satu percobaan yang dilakukan di smartphone Android sms yang dapat dikirim maksimal berisi 10000 karakter. Beberapa smartphone memiliki kapasitas yang berbeda.
  • Pengiriman sms teks base64 diluar kapasitas smartphone memerlukan separasi teks base64 kedalam beberapa sms.
